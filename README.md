🌸 Sweet Greetings by Fares
A beautiful, interactive, and personalized web experience that brightens your day with just a single letter!

✨ About
Sweet Greetings is a minimalist web project designed to share joy. When a user types any letter (a-z) into the input field, a lovely, uplifting message with a cute emoji appears instantly.

Designed and developed by Fares.

🚀 How to Use
Clone this repository to your local machine.

Open index.html in your favorite web browser.

Type any letter on your keyboard.

Enjoy the lovely message!

🛠 Tech Stack
HTML5

CSS3 (Styled with a soft green aesthetic)

JavaScript (For real-time interaction)

🎨 Design
Background: Soft, soothing green.

Interface: Centered, modern card layout with rounded corners.

Vibe: Uplifting, cute, and friendly.

Made with ❤️ by Fares.
